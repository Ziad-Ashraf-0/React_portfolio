## Run Locally

Clone the project

```bash
  git clone https://github.com/Ziad-Ashraf-0/MyReads_Udacity.git
```

Go to the project directory

```bash
  cd MyReads_Udacity
```

Install dependencies

```bash
  npm install
```

Run the project

```bash
  npm start
```
